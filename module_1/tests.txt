- Пользовательский сценарий 1.1 Регистрация пользователя
-- Тестовый сценарий 1.1.1 Регистрация пользователя с валидными данными
- Пользовательский сценарий 1.2 Вход пользователя в аккаунт
-- Тестовый сценарий 1.2.1 Вход зарегистрированного пользователя в аккаунт

Тестовый сценарий 1.1.1
Предусловия:Рандомно сгенерировать валидные данные для входа- адрес эл.почты и пароль.
Шаги:
1.Перейти на форму для регистрации.
2.Ввести адрес электронной почты в поле "Адрес электронной почты".
3.Ввести пароль в поле "Пароль".
4.Ввести пароль в поле "Повторите пароль".
5.Нажать кнопку "Зарегистрировать".
ОР:Пользователь зарегистрирован, отображено собщение на странице:"Спасибо за регистрацию!"

Тестовый сценарий 1.2.1
Предусловия:Подготовить данные для входа,зарегистрированный пользователь - адрес эл.почты и пароль.
Шаги:
1.Перейти на форму для входа.
2.Ввести адрес электронной почты в поле "Адрес электронной почты".
3.Ввести пароль в поле "Пароль".
4.Нажать кнопку "Войти"
ОР:Выполнен вход в аккаунт, отображено собщение на странице:"Рады видеть вас снова!"

- Пользовательский сценарий 2.1 Добавление товара в корзину
-- Тестовый сценарий 2.1.1 Добавление товара в корзину(товар есть на складе)
- Пользовательский сценарий 2.2 Поиск товара
-- Тестовый сценарий 2.2.1 Выполнить поиск товара,который присутствует в каталоге.

Тестовый сценарий 2.1.1
Предусловия:Выбрать товар,который присутствует на складе;вход в аккаунт не выполнен.
Шаги:
1.Зафиксировать название и стоимость товара.
2.Нажать кнопку "Добавить в корзину".
ОР:Товар добавлен, отображены сообщения:"название товара" был добавлен в вашу корзину.Стоимость корзины теперь составляет "стоимость товара".
Доступны действия:посмотреть корзину, оформить.

Тестовый сценарий 2.2.1
Предусловия:Выбрать товар,который присутствует в каталоге;вход в аккаунт не выполнен.
Шаги:
1.Выполнить поиск по полному названию товару.
2.Выполнить поиск по частичному названию товару.
ОР:Товар отображается в списке найденных.
